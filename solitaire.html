Marble Solitaire

This project was for my object oriented design class. The project was made using Java and JavaScript. This is a game of marble solitaire where the user can choose the width of the board and select a peg to jump over another peg into an empty slot to remove the peg that was jumped over. The goal of the game is to only have one peg remaining.

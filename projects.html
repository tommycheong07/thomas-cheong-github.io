Projects

Here are some projects that I have worked on

Spotify Playlist Generator
Spotify API, JavaScript, CSS, HTML, Python
Don't you find that finding new songs for your Spotify playlist is quite the hassle? What if I told you this app changes all that? It will generate a list of songs recommendations based on your top songs and you either like it or dislike it and it will add to your playlist just like Tinder swipes.
Movie Recommendation App
Python, HTML, CSS, PostgreSQL
This movie recommendation app recommends movies to users based on actors and genres selected. Then recommends additional movies that the user may like based on other actors that were in the recommended listings.
Philadelphia Food Delivery Data Visualization
JavaScript, D3, HTML, CSS
Worked alongside a food delivery logistics client to leverage their data to provide insight and analysis on delivery cost efficiency using JavaScript and D3. Created website with data visuals and graphs that enabled restaurants to make better operational decisions that will benefit their profit margins
Marble Solitaire
Java, JavaScript
The project was made using Java and JavaScript. This is a game of marble solitaire where the user can choose the width of the board and select a peg to jump over another peg into an empty slot to remove the peg that was jumped over.
HealthTrak: Virtual Health Assistant
Xcode
This was my first hackathon project in which the theme was a health related hack. This concept app design is intended to help the medical community, especially the patients, to monitor their health and give timely reminders to take medication. As a fallback option, it also alerts family members when the patient's health deteriorates.
